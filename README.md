# 🎮 Number Guessing Game

A Python terminal game where you guess a secret number 
within limited attempts.

## Features
- 3 difficulty levels (Easy, Medium, Hard)
- Hints after every wrong guess
- Score tracking and win streaks
- Input validation

## How to Run
```bash
python number_guessing_game.py
```
